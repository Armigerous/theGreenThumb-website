# The GreenThumb Website

A comprehensive gardening and horticulture website for North Carolina gardeners, built with Next.js, TypeScript, and modern web technologies.

## Features

- **Plant Database**: Comprehensive plant information and search
- **Gardening Tips**: Expert advice and seasonal guidance
- **AI Chat Assistant**: Personalized gardening recommendations
- **User Gardens**: Personal plant tracking and management
- **Responsive Design**: Optimized for all devices

## Logo System

The website supports multiple logo variants for optimal visual integration:

- `logo.png` - Standard logo with background (for social media, metadata)
- `logo-transparent.png` - Logo with transparent background (for hero sections, avatars)

See [Logo System Documentation](./docs/LOGO_SYSTEM.md) for detailed usage instructions.

## Tech Stack

- **Framework**: Next.js 14 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: Shadcn/ui
- **Database**: Supabase
- **CMS**: Sanity
- **Authentication**: Clerk
- **Payments**: Stripe
- **Email**: Resend
- **Analytics**: Vercel Analytics
- **Error Tracking**: Sentry

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Set up environment variables (see `.env.example`)
4. Run the development server: `npm run dev`
